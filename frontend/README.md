# DailyMind Frontend

Next.js 14 frontend for the DailyMind social learning app.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Environment

Create `frontend/.env.local` and set:

```bash
NEXT_PUBLIC_API_URL=http://localhost:5001
```

## Included routes

- `/login`
- `/register`
- `/feed`
- `/create`
- `/profile`
